# JoseJavier_Juarez_trabajoJAVASCRIPT
Trabajo para un curso Java script avanzado

Proyecto final del módulo de **JavaScript Avanzado**, desarrollado como una web corporativa ficticia para la empresa **“Betty Tech Ware”**, dedicada a la venta de productos tecnológicos.

La web incluye:

- Estructura multipágina (Inicio, Galería, Presupuesto, Contacto).
- Carga dinámica de noticias mediante **Ajax (XMLHttpRequest)** desde un archivo JSON externo.
- Galería de productos con filtros, paginación y lightbox.
- Formulario de presupuesto con validaciones avanzadas y cálculo automático.
- Mapa interactivo con cálculo de ruta usando Leaflet y geolocalización.
